UniHelper

A modern PHP-based university helper platform for students.
Note: This is a PHP project and cannot be run directly on GitHub Pages.
Please clone the repo and run it on a local or remote PHP server.

  • Registration & Login with session management
  • Interactive profile with avatar upload
  • Subject management and administration
  • Lecture links and timetable
  • Daily plan and monthly progress tracking
  • Modern, responsive, professional UI
